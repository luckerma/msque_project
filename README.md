# msque_project

Wireless Network Modeling and Simulation

Report available at:\
https://luckerma.github.io/msque_project/

## Install OMNeT++ and INET locally

```bash
pip install opp_env
opp_env install --init -w inet-workspace inet-4.5.4 omnetpp-6.1.0
```

## Run OMNeT++

```bash
cd inet-workspace
opp_env shell
omnetpp
```
